# RST reference

## To add images to the document

```python
.. figure:: <file-name.png>
   :align: center
   :alt: Alternate text description of img

   Alternate-Text-Description-of-IMG
```

## To support "Fancy Notes"
```python
.. note:: <text-string>
```

## To address syntax highlighting
```python
.. highlight:: none
```

## Headings
```python
H1 => # (with overline)
H2 => * (with overline)
H3 => =
H4 => -
H5 => ^
H6 => "
```
