# RST reference


## To add images to the document

```python
.. figure:: <file-name.png>
   :align: center
   :alt: Alternate text description of img

   Fig. <alt-text-name>
```

## To support "Fancy Notes"
```python
.. note:: <text-string>
```

## To address syntax highlighting (really just disable the built-in rendering)
```python
.. highlight:: none
```
